# DevRoast

> Paste your code. Get roasted.

Plataforma para publicar e avaliar código com feedback automático e... nada gentil!🏆 Desenvolvido durante a NLW da Rocketseat.

## O que é o DevRoast?

O DevRoast é uma plataforma onde você pode submeter seu código e receber uma avaliação... digamos, bem honesta. Escolha entre o modo "brutally honest" ou "full roast mode" e veja como seu código se compara aos outros no ranking de vergonha.

## Funcionalidades

- **Editor de Código**: Cole seu código e visualize com destaque de sintaxe
- **Modo Sarcasmo**: Ative o modo "maximum sarcasm enabled" para avaliações mais... condimentadas
- **Sistema de Scoring**: Notas de 0 a 10 que refletem a qualidade (ou falta dela) do seu código
- **Leaderboard**: Veja os piores códigos classificados por vergonha
- **Feedback Detalhado**: Cada submissão recebe uma análise com pontos de melhoria

## Como Usar

1. Acesse a página inicial
2. Cole seu código no editor
3. Ative/desative o modo sarcasmo
4. Clique em "roast_my_code"
5. Receba sua nota e veja como você se sai no ranking

## Tecnologias

- **Next.js 16** - Framework React com App Router
- **TypeScript** - Tipagem estática
- **Tailwind CSS v4** - Estilização utility-first
- **Radix UI** - Componentes acessíveis
- **Shiki** - Highlighting de código

## Getting Started

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/devroast.git

# Instale as dependências
npm install

# Execute o servidor de desenvolvimento
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador.
